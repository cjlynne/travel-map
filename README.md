# 🗺️ 中国旅行足迹地图

一个基于 **Leaflet** 的旅行足迹地图应用，支持真实的中国地图、省份和城市两种模式、标记去过和想去的地点，以及多人合并展示功能。

## ✨ 功能特性

- 🗺️ **真实中国地图**：使用DataV提供的真实GeoJSON地图数据
- 🏷️ **省份标签**：地图上显示每个省份的名称
- 🔍 **两种显示模式**：
  - **省份模式**：点击整个省份进行标记
  - **城市模式**：点击具体城市进行标记
- ✓ **标记去过**：点击省份/城市标记为已去过
- ★ **标记想去**：点击省份/城市标记为想去
- 👥 **多用户支持**：支持多个用户，每个用户有独立的颜色标识
- 🔄 **合并模式**：支持两位用户的旅行足迹合并展示
- 💾 **数据持久化**：数据保存在浏览器本地存储
- 📤 **导入导出**：支持导出和导入JSON数据文件
- 📱 **响应式设计**：支持手机和电脑浏览

## 🚀 快速开始

### 方法1：使用本地服务器（推荐）

由于需要加载外部地图数据，建议使用本地服务器：

```bash
# 进入项目目录
cd travel-map

# 使用Python启动简单服务器
python -m http.server 8000

# 或者使用Node.js
npx http-server

# 或者使用PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

### 方法2：直接打开HTML文件

1. 下载或克隆本项目
2. 直接用浏览器打开 `index.html` 文件
3. 注意：部分功能可能受限于浏览器安全策略

## 📖 使用说明

### 1. 添加用户

1. 点击右上角的「+ 添加用户」按钮
2. 输入用户名
3. 选择用户颜色
4. 点击「确认」按钮

### 2. 切换显示模式

地图右上角有两个按钮：
- **🗺️ 省份**：省份模式，点击整个省份进行标记
- **🏙️ 城市**：城市模式，点击具体城市进行标记

### 3. 标记地点

1. 确保已选中一个用户
2. 选择操作模式：
   - **去过**：标记已去过的地点
   - **想去**：标记想去的地点
3. 点击地图上的省份或城市进行标记
4. 再次点击可以取消标记

### 4. 地图缩放

- 使用鼠标滚轮或触摸板缩放
- 使用地图左上角的 +/- 按钮缩放
- 双击地图可以快速放大

### 5. 合并模式

1. 确保至少有两个用户
2. 点击「合并模式」按钮
3. 选择两位用户
4. 点击「确认合并」

合并模式下的颜色说明：
- **用户颜色**：该用户单独去过
- **紫色**：两人都去过
- **黄色**：有人想去
- **灰色**：都没去过

### 6. 数据管理

- **导出数据**：将所有数据导出为JSON文件
- **导入数据**：从JSON文件导入数据（会覆盖当前数据）
- **合并导入**：从JSON文件导入数据（保留现有用户，添加新用户）
- **重置**：清空所有数据

## 🎨 颜色方案

| 颜色 | 含义 |
|------|------|
| ⚪ 灰色 (#E0E0E0) | 未标记 |
| 🔴 用户颜色 | 该用户去过 |
| 🟡 黄色 (#F39C12) | 想去的地点 |
| 🟣 紫色 (#9B59B6) | 两人都去过（合并模式）|

## 📝 注意事项

1. 数据保存在浏览器本地存储中，清除浏览器数据会丢失所有记录
2. 建议定期使用导出功能备份数据
3. 地图数据来自网络，需要联网才能显示完整地图
4. 城市模式包含主要城市数据（省会和主要城市）
5. 建议使用现代浏览器获得最佳体验
