# 更新日志

## [1.3.0] - 2026-07-15

### 新增
- ✏️ 支持修改用户名（双击用户名即可修改）

## [1.2.0] - 2026-07-15

### 新增
- 🔗 合并导入功能：保留现有用户，添加新用户
- 🔵 用户颜色添加蓝色选项
- 🟡 "想去"颜色改为黄色（省份、城市标记、图例）
- 🔄 合并模式支持城市标记显示

### 修复
- 修复合并导入时用户ID重复的问题
- 修复删除用户时其他用户也被删除的问题
- 修复合并模式下城市标记不显示的问题

## [1.1.0] - 2026-07-15

## [1.0.0] - 2026-07-14

### 初始版本
- 🗺️ 真实中国地图（基于Leaflet和DataV）
- 🏷️ 省份标签显示
- 🔍 省份和城市两种模式
- ✓ 标记去过功能
- ★ 标记想去功能
- 👥 多用户支持
- 🔄 合并模式
- 💾 数据持久化（localStorage）
- 📤 导入导出功能
- 📱 响应式设计
